# 🐍 Snake Game (Python)

A classic **Snake Game** built using **Python** with a graphical user interface powered by **Tkinter**. The game features smooth controls, score tracking, collision detection, and increasing difficulty as the snake grows.

---

## 📌 Features

* 🎮 Classic Snake gameplay
* 🖥️ Interactive GUI using Tkinter
* ⌨️ Keyboard controls (Arrow Keys)
* 🍎 Random food generation
* 📈 Live score display
* 💥 Game Over detection
* 🔄 Restart by running the game again
* ⚡ Smooth movement and collision detection

---

## 🛠️ Technologies Used

* **Python 3.x**
* **Tkinter** – GUI framework
* **Random** – Random food placement
* **Time** *(optional)* – Controls game speed
* **OS** *(optional)* – Platform-related operations

---

## 🎯 Game Rules

1. Control the snake using the arrow keys.
2. Eat the food to increase your score.
3. The snake grows longer after eating food.
4. Avoid colliding with the walls (if enabled).
5. Avoid colliding with your own body.
6. The game ends when a collision occurs.
